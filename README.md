# homebrew-relios

Homebrew tap for [Relios](https://github.com/papa-channy/relios) — a local release pipeline CLI for SwiftPM macOS apps.

## Install

```bash
brew tap papa-channy/relios
brew install relios
```

## Upgrade

```bash
brew update
brew upgrade relios
```

## Requirements

- macOS
- Xcode 15.0 or later (used to build from source)
